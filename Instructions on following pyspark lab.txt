Instructions on following pyspark lab

Use linux/wsl

Install python 3.10
https://askubuntu.com/questions/1539019/how-to-install-python-3-10-on-ubuntu-24-10

Follow this link to setup a virtual environment + pip
https://packaging.python.org/en/latest/guides/installing-using-pip-and-virtual-environments/

Be sure to change any use of python3 into python3.10 to ensure it uses the old version

In the venv install jupyterlab using pip
Then setup a custom kernel so the notebook will use 3.10
python3.10 -m ipykernel install --name py3.10_venv

Finally use this to switch into superuser mode. May need to reenter the venv, cant remember
sudo su


start the notebook as normal:
python3.10 -m jupyterlab [notebook file name here] --allow-root

From here I think all the setup should work ok in the notebook itself.
